* Emacs CRC-32
:PROPERTIES:
  :CUSTOM_ID: emacs-crc-32
:END:
An Elisp library implementing the checksum algorithm Cyclic Redundancy
Check for various bit-widths.

:PROPERTIES:
  :CUSTOM_ID: functions

various info. regarding their specific details, including aliases
